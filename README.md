Sistema de base de datos para administrar una soda/restaurante (pedidos, productos, clientes, etc.).
# Soda Restaurant Database

This project contains the design and implementation of a restaurant database system.

The database manages:

- Customers
- Orders
- Menu items
- Payments

The SQL script creates the tables and relationships required to operate the system.

Tools used:
- SQL Server
- Relational database design
